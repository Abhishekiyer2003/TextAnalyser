# 📝 TextAnalyser Engine (Python CLI Project)

## 📌 Overview

TextAnalyser Engine is a Python-based command-line tool that allows users to perform various text operations on files. It is designed to demonstrate file handling, regular expressions, and basic data processing in Python.

---

## 🚀 Features

* 🔍 Search text in a file using Regular Expressions
* ➕ Add new text to the file
* ✏️ Update (replace) existing text
* ❌ Delete specific text or lines
* 📂 Display file content
* 📊 Track line numbers for search results

---

## 🛠️ Technologies Used

* Python
* File Handling
* Regular Expressions (`re` module)

## ⚙️ How It Works

The program reads a text file and allows users to perform operations such as searching, adding, updating, and deleting text.

### Example Menu:

```
1. Display File Content
2. Search Text
3. Add Text
4. Delete Text
5. Update Text
6. Exit
```

## ▶️ How to Run

1. Clone the repository:

```
git clone <your-repo-link>
```

2. Navigate to the project folder:

```
cd TextAnalyser
```

3. Run the program:

```
python main.py
```
